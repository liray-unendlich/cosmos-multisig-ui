# タスク完了時のチェックリスト

## コード実装完了後に必ず実行するコマンド

### 1. コード品質チェック
```bash
# ESLintでエラーと警告を確認
yarn lint

# 自動修正可能な問題を修正
yarn lint:fix
```

### 2. フォーマット
```bash
# Prettierでコードフォーマット
yarn format
```

### 3. テスト実行
```bash
# テストスイートを実行
yarn test:ci
```

### 4. ビルド確認
```bash
# プロダクションビルドの成功を確認
yarn build
```

### 5. 型チェック
TypeScriptのコンパイルエラーがないことを確認（ビルドで自動実行）

## チェック項目
- [ ] すべてのESLintエラーが解決済み
- [ ] コードがPrettierでフォーマット済み
- [ ] 関連するテストがパス
- [ ] ビルドが成功
- [ ] 不要なconsole.logが削除済み
- [ ] 新しい依存関係が適切に追加済み（package.json）
- [ ] 環境変数が必要な場合は.env.sampleを更新

## コミット前の最終確認
1. `git status`で変更ファイルを確認
2. 不要なファイルがステージングされていないか確認
3. コミットメッセージが変更内容を適切に説明しているか確認
